# Example 086 · Strings

String methods in action: `.len()`, `.trim()`, `.split()` into an array,
`.slice()`, `upper()`/`lower()`, `.contains()`, `.find()` (which returns the
matched text), a `.replace(regex(...), ...)` that collapses whitespace, and
looping over a string as a sequence of characters. Rebuilds a hyphenated
slug into a title. From
[Chapter 16, *Arrays, dictionaries, and strings*](../../book/16-arrays-dictionaries-strings.md).
