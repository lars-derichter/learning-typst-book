# Example 110 · Stack

`#stack` lines content up in one direction with a single `spacing` value.
`dir: ttb` builds a column of full-width rectangles; `dir: ltr` builds a row
of circles. Reach for `stack` when you want a simple one-axis run of items;
reach for `grid` (Chapter 7) when you need aligned rows *and* columns.
Used in [Chapter 21, *Advanced layout*](../../book/21-advanced-layout.md).

```sh
typst compile main.typ out.pdf
```
