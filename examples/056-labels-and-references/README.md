# Example 056 · Labels and references

Numbered headings labelled `<sec:...>` and referenced with `@sec:...`, plus a
figure labelled `<fig:readings>` and referenced with `@fig:readings`. Shows the
`@name` shorthand and the `#ref(<name>)` function side by side. Heading
numbering is on, which references to headings require. From
[Chapter 11, *References and cross-references*](../../book/11-references-and-cross-references.md).

```sh
typst compile main.typ out.pdf
```
