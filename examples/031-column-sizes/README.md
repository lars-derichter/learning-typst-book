# Example 031 · Column sizes

Three kinds of column track in one table: `auto` (as wide as its content),
`1fr` (takes all leftover space and wraps its text), and a fixed `2.5cm`. Used
in [Chapter 7, *Tables and grids*](../../book/07-tables-and-grids.md).

```sh
typst compile main.typ out.pdf
```
