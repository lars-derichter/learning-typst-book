# Example 30 · A basic table

The smallest useful `#table`: one `columns: 3`, then cells listed in reading
order (left to right, top to bottom). Typst wraps to a new row every three
cells. Used in
[Chapter 7, *Tables and grids*](../../book/07-tables-and-grids.md).

```sh
typst compile main.typ out.pdf
```
