# Example 003 · Your first document

The canonical minimal document a beginner compiles from the command line: a
title and a couple of paragraphs of filler text. Walked through in
[Chapter 2, *Getting Typst running*](../../book/02-getting-typst-running.md).

```sh
typst compile main.typ out.pdf
```

Then open `out.pdf`. That whole loop is the entire point of the chapter.
