# Example 28 · Sizing and fit

`fit` modes — `"contain"`, `"cover"`, `"stretch"` — inside identical fixed
boxes so the difference is visible. Reuses the hand-authored `logo.svg`. From
[Chapter 6, *Figures and images*](../../book/06-figures-and-images.md).
