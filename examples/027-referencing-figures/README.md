# Example 027 · Referencing figures

Two labelled figures (`<fig:calm>`, `<fig:spike>`) referenced from prose with
`@fig:calm` and `@fig:spike`. Typst fills in "Figure 1"/"Figure 2" as live,
clickable cross-references that stay correct if the figures move. From
[Chapter 6, *Figures and images*](../../book/06-figures-and-images.md).

```sh
typst compile main.typ out.pdf
```
