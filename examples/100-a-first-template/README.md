# Example 100 · A first template

A minimal `report(body)` function that applies a few set rules and returns the
styled body, invoked with the selector-less `#show: report`. This is the
smallest thing that counts as a template: a function that wraps the whole
document. Used in [Chapter 19, *Templates*](../../book/19-templates.md).

```sh
typst compile main.typ out.pdf
```
