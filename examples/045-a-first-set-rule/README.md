# Example 045 · A first set rule

`#set text` and `#set par` restyle a whole short document from the top: New
Computer Modern at 11pt, justified paragraphs with tighter leading. Nothing
below the rules is styled by hand. Used in
[Chapter 9, *Set rules*](../../book/09-set-rules.md).

```sh
typst compile main.typ out.pdf
```
