# Example 070 · Code vs content blocks

`[...]` produces content you can name, store, and place; `{...}` runs a sequence
of statements and evaluates to their joined result. Shows a content block bound
to a name and a code block that computes then joins two content lines. Used in
[Chapter 13, *From markup to code*](../../book/13-from-markup-to-code.md).

```sh
typst compile main.typ out.pdf
```
