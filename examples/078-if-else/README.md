# Example 078 · if / else

Two faces of the same keyword: a content-producing `#if … [ … ] else [ … ]`
in markup, and an `if` **expression** whose value is stored in a variable with
`#let`. Change `stock` and both the status line and the badge follow. Used in
[Chapter 15, *Control flow*](../../book/15-control-flow.md).
