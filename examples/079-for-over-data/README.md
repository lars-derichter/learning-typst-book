# Example 079 · for over data

Looping to build content: a `for` over an array turns book titles into a
bullet list, and a `for` over a dictionary's `(key, value)` pairs turns office
hours into a term list. Add a book or a day and the output grows itself. Used
in [Chapter 15, *Control flow*](../../book/15-control-flow.md).

```sh
typst compile main.typ out.pdf
```
