# Example 002 · Markup meets code

A five-line `for` loop that generates a list of the powers of two, teasing the
idea that a Typst document is also a small program. Used in
[Chapter 1, *Why Typst?*](../../book/01-why-typst.md).

```sh
typst compile main.typ out.pdf
```
