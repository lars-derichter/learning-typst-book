# Example 24 · Page background

A tinted page `fill` plus a faint, rotated `DRAFT` watermark supplied through
the `background` argument (which draws behind the body text). A `foreground`
would draw on top instead. Used in
[Chapter 5, *Pages and layout*](../../book/05-pages-and-layout.md).

```sh
typst compile main.typ out.pdf
```
