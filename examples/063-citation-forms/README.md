# Example 063 · Citation forms

The same citations shown five ways: parenthetical `@key` (the default), prose
`#cite(<key>, form: "prose")`, `form: "author"`, `form: "year"`, and a page
locator added either in brackets (`@key[p.~262]`) or with the `supplement`
argument. From
[Chapter 12, *Citations and bibliographies*](../../book/12-citations-and-bibliographies.md).

Data file: `refs.yml`.

```sh
typst compile main.typ out.pdf
```
