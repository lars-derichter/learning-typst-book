# Example 083 · Arrays

Building an array and reaching into it: `.len()`, `.first()`/`.last()`,
`.at(i)` with negative indices, `.slice()`, membership with `in`, `.rev()`,
`.push()`/`.pop()` on a mutable binding, and `.sorted(key: ...)`. Also shows
the single-element trailing-comma gotcha (`("x",)` vs `("x")`). From
[Chapter 16, *Arrays, dictionaries, and strings*](../../book/16-arrays-dictionaries-strings.md).
