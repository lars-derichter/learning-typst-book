# Example 062 · An APA reference list

A fuller APA example: five Hayagriva sources of different types (a book, journal
articles with one, two, and three authors, and a web page), a mix of prose and
parenthetical `@key` citations, and a `#bibliography(..., title: [References],
style: "apa")` call. The three-author source shows APA's "et al." rule in-text.
From [Chapter 12, *Citations and bibliographies*](../../book/12-citations-and-bibliographies.md).

Data file: `refs.yml`.

```sh
typst compile main.typ out.pdf
```
