# Example 068 · Values and their types

A literal of each everyday type — integer, float, string, boolean, length,
ratio, content, array, dictionary — shown beside the type `#type(...)` reports,
with `#repr(...)` for the debug view. Ends on the two special values `none` and
`auto`. Used in
[Chapter 13, *From markup to code*](../../book/13-from-markup-to-code.md).

```sh
typst compile main.typ out.pdf
```
