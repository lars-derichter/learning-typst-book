# Example 22 · Paragraph settings

`#set par` with `justify: true`, a tighter `leading` (line spacing), and a
`first-line-indent` (using `all: true` so every paragraph indents). The
book-style look of indented, justified paragraphs. Used in
[Chapter 5, *Pages and layout*](../../book/05-pages-and-layout.md).

