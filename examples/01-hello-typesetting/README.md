# Example 01 · Hello, Typst

The smallest interesting Typst document — a heading, a paragraph, and a bullet
list — showing how little markup it takes to get a real typeset page. Used in
[Chapter 1, *Why Typst?*](../../book/01-why-typst.md).

```sh
typst compile main.typ out.pdf
```
