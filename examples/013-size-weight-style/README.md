# Example 013 · Size, weight, and style

The three everyday axes of type: `size` in points and ems, `weight` by name and
by number (100–900), and `style` (normal / italic / oblique), with the markup
shortcuts `*strong*` and `_emph_`. Used in
[Chapter 4, *Text and fonts*](../../book/04-text-and-fonts.md).

```sh
typst compile main.typ out.pdf
```
