# Example 058 · Table of contents

A multi-heading document with `#outline(title: [Contents], depth: 2)`. The
`depth: 2` keeps the level-3 heading out of the contents. A small fixed page
plus `#lorem` filler spreads the body over several pages, so the outline shows
real page numbers. From
[Chapter 11, *References and cross-references*](../../book/11-references-and-cross-references.md).

```sh
typst compile main.typ out.pdf
```
