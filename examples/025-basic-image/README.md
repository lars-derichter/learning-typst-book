# Example 025 · Placing an image

Loads a small hand-authored SVG (`logo.svg`, plain text so it stays diffable)
with `#image`, first at its natural size and then scaled to a fixed width, with
`alt` text for accessibility. From
[Chapter 6, *Figures and images*](../../book/06-figures-and-images.md).

```sh
typst compile main.typ out.pdf
```
