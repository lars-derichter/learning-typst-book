# Example 080 · range and while

`range(5, 55, step: 5)` counts in fives and stops before 55 (ranges are
half-open). A `while` loop doubles a value until it passes 1000 — the same
powers of two from Chapter 1, this time as a loop with a running variable.
Used in [Chapter 15, *Control flow*](../../book/15-control-flow.md).
